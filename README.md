# Trinket Cooldown Standalone

**Version:** 1.5 
**Author:** Bro_Raz

---

## Description

Trinket Cooldown Standalone is a World of Warcraft addon (tested on Midnight 12.0 prepatch) that tracks your equipped trinkets' cooldowns.  

Features include:

- Tracks on-use trinkets only
- Desaturates icon when on cooldown; full color when ready.  
- Cooldown timer text on icons.  
- Movable, scalable frame with vertical or horizontal layout.  
- Config window with lock/unlock, padding, scale sliders and as of V1.5 opacity slider also 
- Slash commands: `/trinkets` or `/stt` to open settings.

---

## Installation

1. Place the folder `TrinketCooldownStandalone` into your `World of Warcraft/_retail_/Interface/AddOns/` directory.  
2. Make sure it contains:
    - `TrinketCooldownStandalone.lua`
    - `TrinketCooldownStandalone.toc`  
    - (Optional) `README.md` 
3. Reload the UI or restart WoW.  
4. Use `/trinkets` or `/stt` to open the settings window.
